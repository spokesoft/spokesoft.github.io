# Spokesoft.github.io

Spokesoft github pages site.

<br />

## Documentation

### What you will need to build

- npm
- gulp

<br />

### How to build

```
gulp build:all
```

or to clean the project

```
gulp clean:all
```

or to watch files and build on changes

```
gulp watch:all
```
